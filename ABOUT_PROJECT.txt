1. ПРИЗНАЧЕННЯ ПРОГРАМИ
------------------------
Програма призначена для автоматизованого збору та структурування
інформації з новинних веб-сторінок за допомогою великих мовних
моделей (LLM). На відміну від традиційних скраперів, які потребують
ручного налаштування CSS-селекторів, ця система використовує
DeepSeek для семантичного розуміння контенту.


2. АРХІТЕКТУРА
--------------
  [Користувач] → Streamlit UI (ui.py) → FastAPI API (main.py)
                                         → Crawl4AI + Playwright
                                         → DeepSeek LLM
                                         → SQLite (історія)

  - ui.py       — веб-інтерфейс (Streamlit)
  - main.py     — бекенд API (FastAPI)
  - scraper.py  — модуль скрапінгу (Crawl4AI + DeepSeek)
  - database.py — робота з БД (SQLite, bcrypt)
  - i18n.py     — український та англійський переклад інтерфейсу


3. ЯК ЦЕ ПРАЦЮЄ (покроково)
----------------------------
1. Користувач вводить URL новинної статті у веб-інтерфейсі
2. Запит надсилається на бекенд (FastAPI)
3. Бекенд запускає Crawl4AI з AsyncWebCrawler
4. Playwright (Chromium) завантажує сторінку
5. LLMExtractionStrategy з DeepSeek аналізує вміст:
   - визначає заголовок, автора, дату, зображення
   - визначає тип статті (новина, інтерв'ю, думка)
   - якщо інтерв'ю — витягує ключові цитати
   - перекладає на обрану мову (якщо потрібно)
6. Структуровані дані повертаються у JSON
7. Результат зберігається в історію (SQLite)
8. Користувач бачить картку з інформацією


4. АДАПТИВНІСТЬ (ключова ідея диплому)
----------------------------------------
Замість жорстких CSS-селекторів (які ламаються при зміні дизайну),
програма використовує LLM для смислового аналізу сторінки:

  - Стійкість до змін верстки — LLM аналізує зміст, не HTML
  - Універсальність — працює на будь-якому новинному сайті
  - Контекст — розуміє що "Ілон Маск" і "засновник Tesla" те саме
  - Тип статті — автоматично визначає новина/інтерв'ю/думка


5. СТЕК ТЕХНОЛОГІЙ
--------------------
  Python 3       — мова програмування
  FastAPI        — REST API (uvicorn сервер)
  Crawl4AI       — веб-скрапінг з Playwright (Chromium)
  DeepSeek       — LLM для витягування та перекладу даних
  Streamlit      — веб-інтерфейс
  SQLite         — база даних (користувачі, історія)
  bcrypt         — хешування паролів


6. МОЖЛИВОСТІ
--------------
  1. Адаптивне витягування даних через LLM (без селекторів)
  2. Автовизначення типу статті (інтерв'ю → ключові цитати)
  3. Переклад контенту (UA/EN) з індикацією перекладу
  4. Реєстрація/вхід та персональна історія скрапінгу
  5. Експорт історії у JSON та CSV
  6. Двомовний інтерфейс (українська/англійська)
