Програма призначена для автоматизованого збору та структурування інформації з новинних веб-сторінок за допомогою великих мовних моделей (LLM). На відміну від традиційних скраперів, які потребують ручного налаштування CSS-селекторів, ця система використовує DeepSeek для семантичного розуміння контенту.
